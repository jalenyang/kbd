# kbd

kubernetes based development

## One example

```shell
$ kbd --help                                           
Usage:  kbd [OPTIONS] COMMAND
kubernetes based development toolkit                              
Options:                                                          
-d      enable debug mode                                 
Commands:                                                         
e2e     manage the e2e tests in the kubernetes cluster    
realm   manage the realms in the kbd                      
helm    manage the helm business in the kubernetes cluster
```
